# FastAPI WebSocket Chat

A real-time chat application built while learning WebSockets using FastAPI and vanilla JavaScript.

## Features

- Real-time messaging
- Username registration
- Online user count
- Join/Leave notifications
- Server-managed usernames

## Tech Stack

- FastAPI
- WebSockets
- HTML
- CSS
- JavaScript

## Future Improvements

- Duplicate username detection
- Typing indicator
- Private messaging
- Chat rooms
- Message history
- Authentication
